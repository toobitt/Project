loadUrl(object param)

方法
功能描述：加载一段外链或内链
场       景：需要跳转页面的操作
所属模块： dingdone

外内链规则

外内链接的规则

部分跳转功能需客户端支持该功能，否则，跳转的动作将无任何的效果

规则: url = 外链地址

支持http 和 https 跳转

例如: http://www.baidu.com

跳转到内容的详情

规则: [type|mid|cid]

  • type: 内容的类型 [news | vod | tuji | vote] [图文混排 | 视频 | 图集 | 投票]
  • mid: 模块的ID
  • cid: 详情的ID

例如: news|12898|99878 , 跳转到12898模块下 图文数据 99878

跳转到用户模块

规则: [mid#x]

  • mid: 模块的ID

例如: 12898#x, 跳转到模块12898模块的首届面

跳转到内置模块

规则: [type#]

  • userLogin: 用户登录的界面
  • userCenter: 用户中心的界面
  • weather: 天气的界面
  • aboutUs: APP关于我们界面 (dev)
  • feedBack: APP的意见反馈界面 (dev)

例如： userLogin#， 跳转到用户的登录界面

参数列表
参数名 类型 描述
param object {
    url:xxxxxxx,    //需要跳转的链接地址
}

客户端返回码说明

说明
code 说明
0 处理成功
10 URL无法校验,NetError
11 URL无权限使用SDK,不在白名单中
12 URL无权限执行当前API方法
13 当前执行的方法客户端不支持，需升级客户端
14 客户端执行方法时出现异常
